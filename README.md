# Погодный Telegram бот

Простой Telegram бот для получения прогноза погоды от OpenWeatherMap.

## Возможности

- 🌤 Текущая погода в любом городе мира
- 🌡 Подробная информация: температура, влажность, ветер, давление
- 🎛 Удобная кнопка для запроса погоды
- 🌍 Поддержка городов на русском и английском
- 😊 Эмодзи под настроение погоды

## Команды

| Команда | Описание |
|---------|----------|
| `/start` | Начать работу с ботом |
| `/weather [город]` | Узнать погоду |
| `/help` | Показать справку |
